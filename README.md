# 🌾 CropCart - Farmer Marketplace App

CropCart is a simple Android application designed to connect farmers directly with customers. Farmers can list their natural products, and customers can browse and purchase them easily.

This project was built as a showcase application for learning and demonstrating Android development skills.

---

## 📱 Features

### 👨‍🌾 Farmer Panel
- Add products (fruits, vegetables, cereals, pulses, oil seeds, spices, seeds, etc.)
- Manage product listings

### 🛒 Customer Panel
- Browse available products
- Add items to cart
- Provide delivery address

### 🔐 Authentication
- Email and password-based login using Firebase Authentication

### 🌦️ Additional Features
- Weather updates using API
- News updates related to agriculture
- Gemini API integration for enhanced functionality

---

## 🛠️ Tech Stack

- **Language:** Kotlin  
- **UI:** XML  
- **Backend & Database:** Firebase Realtime Database  
- **Authentication:** Firebase Auth  
- **APIs Used:**  
  - Weather API  
  - News API  
  - Gemini API
